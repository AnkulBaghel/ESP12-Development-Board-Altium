# ESP12-Development-Board-Altium
Custom ESP8266 Development Board designed in Altium Designer

## Overview

This project is a custom ESP8266 Development Board designed using Altium Designer.

## Features

- ESP12 Module
- CH340C USB to UART
- AMS1117 3.3V Regulator
- USB Programming
- Reset Circuit
- Power LED
- User LED
- GPIO Headers
- 12V Input
- 50mm × 70mm PCB

## Software

- Altium Designer 26

## Manufacturing Files

- Schematic
- PCB Layout
- Gerber
- Drill Files
- BOM
